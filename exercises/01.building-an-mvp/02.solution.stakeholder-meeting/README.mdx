# Stakeholder Meeting

👨‍💼 Great work. You used stakeholder input to transform a prep outline into a
decision-ready discovery brief for cross-organization social coordination.

🦉 A strong result is not "more notes." It is clearer scope, measurable success
criteria, and known risks with explicit next actions.

Before locking scope, include a quick viability checkpoint: "Should we build
this at all, defer it, or buy/adopt an existing solution?" A good meeting
result captures why existing options fall short for this use case and what
distinct advantage this MVP can create.
